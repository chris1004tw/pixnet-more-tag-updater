一鍵將痞客邦 More 標籤轉換為 WordPress 格式。

== Description ==

您可以透過這個外掛，一件將痞客邦「繼續閱讀 (More)」標籤轉換為 WordPress 格式，不需要透過手動 SQL 更新。

**Plugin Features**

* 一鍵更新所有來自痞客邦的 More 標籤
* 不需要手動執行 SQL 更新
* 轉換完成後即可移除外掛

== Installation ==

1. 備份資料庫
1. 下載並安裝本外掛
1. 進入 工具 > 痞客邦 More 轉換 頁面
1. 點擊「開始轉換」
1. 完成
