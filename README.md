# 痞客邦 More 標籤更新工具
將痞客邦 More 標籤轉換成 WordPress 格式。

## 介紹
**痞客邦**: ```<!-- more -->```

**WordPress**: ```<!--more-->```

只要點一下，這個工具就會幫您轉換痞客邦「繼續閱讀 (More)」標籤成 WordPress 格式。

## 安裝
1. [點這裡](https://github.com/kocpc/pixnet-more-tag-updater/releases)下載這個外掛。
2. 上傳到 WordPress 後台並安裝。
3. 備份您的資料庫。
3. 進入 工具 > 痞客邦 More 轉換 頁面。
4. 點擊「開始轉換」。
5. 完成！
